# 📖 To Do App <a name="about-project"></a>

**To Do App** is a regular usable app with options to add To Do's, mark them as complete and delete them after it's completed.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:


Example command:

```sh
 npm install
```


### Setup

Clone this repository to your desired folder:


Example commands:

```sh
  cd my-folder
  git clone https://github.com/sumon766/to-do-react.git
```

### Install

Install this project with:

Example command:

```sh
  cd my-project
  npm install or yarn install
```

### Usage

To run the project, execute the following command:


Example command:

```sh
  npm run build

  npm start
```
